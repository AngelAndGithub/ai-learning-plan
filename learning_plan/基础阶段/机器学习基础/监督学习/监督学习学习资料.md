## 一、监督学习概述

### 1.1 什么是监督学习

监督学习是机器学习的一种方法，其中模型从标记的训练数据中学习，以预测新的、未标记的数据。监督学习的核心是建立输入特征与输出标签之间的映射关系。

### 1.2 监督学习的类型

- **分类**：预测离散的类别标签
- **回归**：预测连续的数值
- **序列预测**：预测时间序列或序列数据

### 1.3 监督学习的工作流程

1. 数据收集与预处理
2. 特征工程
3. 模型选择与训练
4. 模型评估与调优
5. 模型部署与监控

## 二、线性回归

### 2.1 线性回归的基本概念

线性回归是一种用于预测连续变量的监督学习算法，它假设输入特征与输出之间存在线性关系。

### 2.2 简单线性回归

简单线性回归涉及一个自变量和一个因变量，模型形式为：

$$ y = eta_0 + eta_1 x + psilon $$

其中：
- $y$ 是因变量
- $x$ 是自变量
- $eta_0$ 是截距
- $eta_1$ 是斜率
- $psilon$ 是误差项

### 2.3 多元线性回归

多元线性回归涉及多个自变量和一个因变量，模型形式为：

$$ y = eta_0 + eta_1 x_1 + eta_2 x_2 + ... + eta_n x_n + psilon $$

### 2.4 线性回归的求解方法

- **最小二乘法**：最小化残差平方和
- **梯度下降法**：通过迭代优化参数

### 2.5 线性回归的评估指标

- **均方误差（MSE）**：$MSE = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2$
- **均方根误差（RMSE）**：$RMSE = \sqrt{MSE}$
- **平均绝对误差（MAE）**：$MAE = \frac{1}{n} \sum_{i=1}^{n} |y_i - \hat{y}_i|$
- **R² 评分**：$R^2 = 1 - \frac{\sum_{i=1}^{n} (y_i - \hat{y}_i)^2}{\sum_{i=1}^{n} (y_i - \bar{y})^2}$

## 三、逻辑回归

### 3.1 逻辑回归的基本概念

逻辑回归是一种用于分类任务的监督学习算法，它使用 sigmoid 函数将线性组合转换为概率。

### 3.2 逻辑回归的模型形式

$$ P(y=1|x) = \frac{1}{1 + e^{-(eta_0 + eta_1 x_1 + ... + eta_n x_n)}} $$

### 3.3 逻辑回归的损失函数

- **对数损失函数**：$L(y, \hat{y}) = -[y \log(\hat{y}) + (1-y) \log(1-\hat{y})]$

### 3.4 逻辑回归的求解方法

- **梯度下降法**：通过迭代优化参数
- **牛顿法**：利用二阶导数加速收敛

### 3.5 逻辑回归的评估指标

- **准确率**：$Accuracy = \frac{TP + TN}{TP + TN + FP + FN}$
- **精确率**：$Precision = \frac{TP}{TP + FP}$
- **召回率**：$Recall = \frac{TP}{TP + FN}$
- **F1 分数**：$F1 = 2 \times \frac{Precision \times Recall}{Precision + Recall}$
- **ROC 曲线**：接收器操作特征曲线
- **AUC**：ROC 曲线下的面积

## 四、决策树

### 4.1 决策树的基本概念

决策树是一种基于树结构的监督学习算法，它通过一系列的决策规则将数据分类或回归。

### 4.2 决策树的构建过程

1. 选择最佳特征进行分裂
2. 递归地构建子树
3. 剪枝以防止过拟合

### 4.3 决策树的分裂标准

- **分类树**：信息增益、增益率、基尼系数
- **回归树**：均方误差、平均绝对误差

### 4.4 决策树的优点和缺点

**优点**：
- 易于理解和解释
- 能够处理非线性关系
- 不需要特征归一化

**缺点**：
- 容易过拟合
- 对噪声敏感
- 可能产生不稳定的树结构

## 五、随机森林

### 5.1 随机森林的基本概念

随机森林是一种集成学习算法，它由多个决策树组成，通过投票或平均来提高预测性能。

### 5.2 随机森林的构建过程

1. 从原始数据中随机采样（有放回）
2. 对每个样本选择随机子集特征
3. 构建多个决策树
4. 综合所有树的预测结果

### 5.3 随机森林的优点和缺点

**优点**：
- 减少过拟合
- 提高预测准确性
- 能够处理高维数据

**缺点**：
- 计算成本较高
- 模型解释性较差

## 六、支持向量机

### 6.1 支持向量机的基本概念

支持向量机（SVM）是一种用于分类和回归的监督学习算法，它通过寻找最优超平面来分隔不同类别的数据。

### 6.2 线性支持向量机

线性 SVM 寻找能够最大化类别间间隔的超平面。

### 6.3 非线性支持向量机

通过核函数将数据映射到高维空间，从而实现非线性分类。

### 6.4 常用的核函数

- **线性核**：$K(x, x') = x^T x'$
- **多项式核**：$K(x, x') = (x^T x' + c)^d$
- **高斯核（RBF）**：$K(x, x') = e^{-\gamma ||x - x'||^2}$

### 6.5 支持向量机的优点和缺点

**优点**：
- 在高维空间中表现良好
- 能够处理非线性关系
- 对小样本数据有效

**缺点**：
- 计算成本较高
- 对参数选择敏感
- 对噪声敏感

## 七、模型评估与调优

### 7.1 交叉验证

- **K 折交叉验证**：将数据分为 K 份，每次用 K-1 份训练，1 份测试
- **留一交叉验证**：K 等于样本数量
- **留 P 交叉验证**：每次留出 P 个样本作为测试集

### 7.2 超参数调优

- **网格搜索**：遍历所有可能的参数组合
- **随机搜索**：随机采样参数组合
- **贝叶斯优化**：基于贝叶斯定理优化参数

### 7.3 模型选择

- **偏差-方差权衡**：选择适当复杂度的模型
- **AIC/BIC**：基于信息准则选择模型
